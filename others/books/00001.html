**前端（Front-end）** 负责用户直接看到和交互的部分——即**浏览器端**呈现的页面界面和交互体验。 - **后端（Back-end）** 负责服务器端的业务逻辑、数据处理和存储——即“看不见但支撑着一切”的部分。 打个比方：如果你走进一家餐厅就餐—— - **前端**就如同餐厅的用餐环境、菜单设计、服务员的服务——这些都是顾客（用户）能直接看到和感受到的。 - **后端**就如同后厨的厨师、食材采购、财务结算——顾客看不到，但菜品的美味和餐厅的正常运营离不开它们。 **Web前端开发**，就是专门负责Web应用中用户可见部分的设计与开发工作——包括页面结构的搭建、视觉样式的实现、用户交互逻辑的编写、页面性能的优化等。 随着移动互联网的普及，Web前端开发的范畴已经从传统的桌面网站，扩展到了移动端网页、微信小程序、混合移动应用（Hybrid App）、桌面跨平台应用（如Electron）等多个领域。可以说，**只要有图形界面的地方，就有前端开发者的身影**。 ### 1.1.2 Web前端开发的三大核心技术 Web前端开发的核心技术有三项：**HTML、CSS、JavaScript**。这三者分工明确、协同配合，共同构成了现代Web应用的基石。 我们可以用**人体结构**来形象地理解三者的关系： | 技术 | 类比 | 作用 | |------|------|------| | **HTML** | 骨骼结构 | 定义网页的内容和结构 | | **CSS** | 皮肤与衣装 | 控制网页的样式与布局 | | **JavaScript** | 神经系统与肌肉 | 实现网页的交互与动态效果 | 分开来看： #### 1. HTML——超文本标记语言 HTML（HyperText Markup Language）是用来**描述网页结构和内容**的标记语言。它通过一系列**标签（Tag）** 来标记文本、图像、链接、表格、表单等不同类型的页面元素，告诉浏览器“这里应该显示一个标题”“那里应该放一张图片”“此处是一个可点击的按钮”。 示例1-1：一个最简单的HTML片段 ```html

欢迎来到我的网站

这是我的第一个网页段落。

点击这里访问示例网站 ``` 在上面的代码中： - `

`标签表示**一级标题** - `

`标签表示**段落** - ``标签表示**超链接** 浏览器会读取这些标签，并按照默认样式将它们渲染成可视化的页面元素。 HTML是Web的**基础语言**——没有HTML，就没有网页的“骨架”，CSS和JavaScript也就失去了施展的空间。 #### 2. CSS——层叠样式表 CSS（Cascading Style Sheets）是用来**控制HTML元素呈现样式**的语言。它负责网页的“美颜”——包括颜色、字体、大小、间距、位置、动画效果等。 示例1-2：CSS代码示例 ```css h1 { color: blue; /* 标题变为蓝色 */ font-size: 36px; /* 标题字号为36像素 */ text-align: center; /* 标题居中显示 */ } p { color: #333; /* 段落文字为深灰色 */ line-height: 1.8; /* 行高为1.8倍 */ } ``` 有了CSS，原本单调乏味的纯文本页面可以变得色彩丰富、布局精致、赏心悦目。 #### 3. JavaScript——交互脚本语言 JavaScript（简称JS）是一种**脚本编程语言**，用于为网页**添加交互行为和动态功能**。它可以响应用户的点击、输入、滚动等操作，动态更新页面内容，与服务器进行数据通信（Ajax），实现复杂的业务逻辑等。 示例1-3：JavaScript代码示例 ```javascript // 当用户点击按钮时，弹出一个欢迎对话框 document.getElementById('myButton').onclick = function() { alert('欢迎你，亲爱的读者！'); }; // 动态修改页面内容 document.querySelector('h1').textContent = '你已成功修改了标题！'; ``` 有了JavaScript，网页不再是“静态的印刷品”，而是一个**可以实时响应用户操作的“应用”**。 #### 三者协同工作 下面通过一个具体的例子来说明三者如何协同工作： | 职责 | 代码示例 | 实现效果 | |------|---------|---------| | **HTML（结构）** | `` | 在页面上显示一个按钮 | | **CSS（样式）** | `button { background: red; color: white; }` | 按钮变为红底白字 | | **JavaScript（行为）** | `button.onclick = function() { ... }` | 点击按钮时弹出提示 | 在真实的开发中，这三者通常分别写在独立的文件中（.html、.css、.js），各司其职、清晰维护——这就是**结构、样式、行为分离**的开发原则。 > **💡 常识补充**：JavaScript虽然名字中带有“Java”，但它和Java编程语言是完全不同的两种语言，只是在命名上借用了当时如日中天的Java的名气来提升知名度。 ### 知识小结 本节我们学习了： 1. **Web前端**是用户直接面对和交互的Web界面部分。 2. **HTML、CSS、JavaScript**是Web前端开发的三大核心技术，分别负责**结构、样式、行为**。 3. 三者协同配合，共同构成完整的Web应用。 ### 思考与练习 **一、选择题** 1.
